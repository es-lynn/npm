# Setup

`yarn setup`

# Running tests

`yarn test:node`
 - Runs all tests inside `node` runtime
 
 `PKG=eslynn-utils yarn test:node`
 - Runs tests only for the package `eslynn-utils`

# Publishing

create a branch named `next/<package-name>` and push
